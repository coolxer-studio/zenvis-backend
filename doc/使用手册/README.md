# 产品手册

> ZenVis是一款面向企业级用户的专业全链路观测管控平台，致力于帮助企业有效管理信息技术资产，提供从数据采集、资产梳理、运营分析到安全监控的一体化服务。平台通过多端点数据采集与智能分析，显著提升IT资产管理效率与安全性。

### 产品概述

ZenVis平台基于先进的AI大模型技术和大数据分析能力，为用户提供全方位的智能化服务。平台支持Android、iOS、Web、小程序、主机、容器、中间件、智能终端等多种平台的监测，能够应对日益复杂的应用与业务安全挑战和监管合规要求。

平台采用模块化、插件化的架构设计，具备强大的扩展能力，用户可以根据业务需求定制和扩展系统功能，实现业务场景的快速构建和部署。

### 核心功能

1. **大屏看板**
   - 支持按照客户需求定义增加自定义大屏
   - 提供直观的数据可视化展示
   - 实时监控系统状态和关键指标

2. **全局检索**
   - 支持跨模块、跨数据源的统一检索
   - 快速定位所需日志数据
   - 支持多维度、多指标的检索

3. **数智中心**
   - 基于AI大模型实现自然语言交互，降低用户使用门槛
   - 提供知识问答、智能巡检、研判分析、策略控制等智能化服务
   - 支持自动化处理从数据查询到风险处置的全流程

4. **插件应用系统**
   - 模块化设计，支持热插拔
   - 可根据特定业务需求定制和扩展系统功能

5. **资产管理【插件】**
   - 自动化整合硬件类、软件类和数据类资产
   - 提供资产归类、标记、上下线管理、重要性评定和风险等级评定
   - 构建全面、清晰且有序的数据资产视图

6. **运营分析【插件】**
   - 行为事件：跟踪和分析用户在应用或网站上的行为，如点击、滑动、输入等
   - 状态指标：展示应用的关键性能指标，如资源利用率、地理位置分布等
   - 应用质量：监控应用的质量，包括崩溃率、错误率等


7. **风险管控【插件】**
   - 脆弱性检测
   - 威胁监测
   - 风险管理

### 技术优势

- **多平台支持**：全面支持移动端（Android、iOS）、Web端、主机端、容器、中间件、智能终端等多种平台
- **AI驱动**：融合AI大模型技术，提供智能化分析和服务能力
- **插件化架构**：采用插件化设计，支持灵活的功能扩展
- **全流程覆盖**：涵盖数据采集、分析、策略制定、执行验证等完整流程
- **安全合规**：满足国家网络安全法、等级保护等合规要求

### 适用场景

- **企业IT资产管理**：帮助企业管理分散的IT资产，实现资产的动态梳理和统一视图
- **安全监控与威胁识别**：实时监测安全威胁，识别潜在风险并提供处置建议
- **运营分析与决策支持**：基于数据分析提供资源规划、成本控制等优化建议
- **合规审计**：满足监管机构的合规要求，提供完整的审计和报告功能

### 核心价值

- **提升管理效率**：通过自动化手段大幅提升IT资产管理效率
- **增强安全性**：实时监控和分析安全威胁，降低安全风险
- **降低运营成本**：智能化处理减少人工干预，降低人力成本
- **辅助决策**：通过数据分析和模型预测，为业务决策提供科学依据
