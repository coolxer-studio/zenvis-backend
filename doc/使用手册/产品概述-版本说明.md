# 版本说明

## 版本支持

| 服务     | 基础版                                     | 专业版                                         | 定制化                                     |
| -------- | ------------------------------------------ | ---------------------------------------------- | ------------------------------------------ |
| 部署方式 | Saas                                       | Saas、私有化                                   | 私有化                                     |
| 探针平台 | Android、Ios、H5                           | Android、Ios、H5                               | Android、Ios、H5                           |
| 数据安全 | 通信保护                                   | 通信保护+自定义密钥                            | 通信保护+自定义密钥                        |
| 设备标签 | 通用标签、事件标签（固定）、自定义标签     | 通用标签、事件标签（私有化可配置）、自定义标签 | 通用标签、事件标签（可配置）、自定义标签   |
| 数据周期 | 一个月                                     | 一年（私有化可配置）                           | 一年（可配置）                             |
| 日活支持 | 1万                                        | 按需支持                                       | 按需支持                                   |
| 仪表板   | 支持（无自定义）                           | 支持（可自定义）                               | 支持（可无自定义）                         |
| 数据检索 | 支持（简单搜索）                           | 支持（简单搜索+高级搜索）                      | 支持（简单搜索+高级搜索）                  |
| 安全分析 | 安全事件（内置规则）、设备指数（内置规则） | 安全事件（配置规则）、设备指数（配置规则）     | 安全事件（配置规则）、设备指数（配置规则） |
| 数据管理 | ✗                                          | ✓（部分）                                      | ✓                                          |
| 策略配置 | ✗                                          | ✓（部分）                                      | ✓                                          |
| 系统配置 | ✗                                          | ✗                                              | ✓                                          |
| 风险指数 | ✓                                            | ✓                                              | ✓                                          |
| 设备指纹 | ✗                                          | ✓                                              | ✓                                          |
| 关联关系 | ✗                                          | ✓                                              | ✓                                          |
| 定制需求 | ✗                                          | ✓（部分）                                      | ✓                                          |
